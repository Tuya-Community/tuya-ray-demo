## 电量统计模版

- 产品名称：电量统计模板

### 快速开始：

1. 前往[IoT 平台](https://iot.tuya.com/)创建产品

2. 选择品类 `插座`
   ![功能选择](./images/iot01.png)
3. 功能选择

   必须的功能点

   ```
   switch_1,
   countdown_1,
   add_ele
   ```

4. 安装项目并启动项目

   ```
   yarn && yarn start
   ```

## 注意

此模板需要产品开启电量统计高级能力，才能正常使用电量统计功能。
统计接口中 type 参数的相关说明可参考[这里](https://www.tuyaos.com/viewtopic.php?t=973)
