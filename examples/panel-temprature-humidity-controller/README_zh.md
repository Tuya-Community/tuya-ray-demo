## 2、 快速上手

## 3、能力依赖

- App 版本
  - 涂鸦智能 5.1.0 及以上版本
- TTT 依赖
  - BaseKit: 3.11.13
  - MiniKit: 3.9.8
  - DeviceKit: 4.5.1
  - BizKit: 4.5.1
  - baseversion: 4.5.1
- 组件依赖
  - [Smart UI](https://developer.tuya.com/material/smartui?comId=help-getting-started)
  - [Common Charts](https://developer.tuya.com/material/library_oHEKLjj0/component?code=CommonCharts)
- 功能页依赖
  - 暂无

## 4、面板功能

- 温湿度统计
- 设置
